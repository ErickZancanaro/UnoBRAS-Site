O que são sistemas embarcados?

Um sistema embarcado é um computador compacto e especializado, projetado para realizar uma única tarefa específica dentro de um dispositivo maior.
Diferente de um computador comum, ele não serve para múltiplos propósitos

Qual a utilidade?

Um sistema embarcado serve para automatizar, controlar ou monitorar uma função específica dentro de um aparelho, tornando-o inteligente e independente.
Em vez de usar engrenagens mecânicas complexas, usa-se um chip com programação e assets para criar os aparelhos, como irrigadores automáticos, luzes noturnas, robôs, etc.
