BIBLIOTECAS ÚTEIS:

Bibliotecas Úteis: Integrações de comandos facultativos no código para uso de algum componente ou função. No C++ (ARDUINO) elas sempre são acompanhadas por um "#include".

No Arduíno, muitas bibliotecas ja vem integradas no código e não precisam ser chamadas, como o buzzer, por exemplo. Contudo, grande parte dos assets precisam que suas bibliotecas úteis sejam inseridas.

COMPONENTES E SUAS BIBLIOTECAS:

Servo Motor

Setup:
#include < Servo.h >

Comandos:

  1. Servo n; // sendo n o nome que você quer dar para o servo;
  2. servo.attach // pino;
  3. servo.write // rotação;

Display LCD

Setup:
#include < LiquidCrystal_I2C.h >

Comandos:

  1. LiquidCrystal_I2C lcd(0x27, 16, 2) // Define a resolução;
  2. lcd.init() // chama o lcd;
  3. lcd.backlight() // ativa os leds;
  4. lcd.setCursor(X, Y) // X = quantos slots serão pulados até o texto começar e Y = linha do texto;
  5. lcd.print("Texto") // O que aparecerá no display;
  6. lcd.clear() // limpa a mensagem;
