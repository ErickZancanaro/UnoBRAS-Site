C++:

Variáveis

As variáveis são declaradas no topo do código, sendo a estrutura:

Declaração das variáveis aqui!

int a // valor inteiro;

const int b // valor inteiro constante (Não muda);

bool c // valor booleano (sim ou não);

void setup()
{
// setup
}

void loop()
{
// código
}

Void Setup

O void setup é a configuração das variáveis, pinos, assets, etc. A função consiste em:

void setup()
{
pinMode(LED, OUTPUT); // pinMode = Configura o pino ; LED = pino X ; OUTPUT = Saída --- Aqui é configura o pino LED como saída de energia
}

void loop()
{
// código
}

Void Loop

O void Loop é onde o código deve ficar:

LED = (X) // variável LED está ligada no pino X

void setup()
{
pinMode(LED, OUTPUT);
}

void loop() // aqui fica todo o código de programação.
{
// código
}

Comandos

Os principais controles do void loop são: :

void loop()
{

digitalRead // Ler pino digital X

digitalWrite // Escrever para o pino digital X

analogRead // Ler pino analógico X

analogWrite // Escrever para o pino analógico X (Exclusivo para entradas PWM "~" ou "-")

delay(seconds) // Atraso

}
